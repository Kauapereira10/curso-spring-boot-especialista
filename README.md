# 🚀 Spring Boot - Do Iniciante ao Profissional

## 📖 Sobre o Repositório

Este repositório foi criado para registrar minha evolução no ecossistema Spring Boot durante um curso completo de desenvolvimento backend com Java.

Ao longo dos estudos, adicionarei exemplos práticos, projetos, desafios, anotações e implementações das principais tecnologias utilizadas no mercado para construção de aplicações modernas, escaláveis e seguras.

O objetivo é consolidar conhecimento, praticar boas práticas de desenvolvimento e construir um portfólio sólido para oportunidades profissionais na área de desenvolvimento Java.

---

## 🎯 Objetivos

* Aprender o ecossistema Spring Framework
* Desenvolver APIs RESTful profissionais
* Trabalhar com bancos de dados utilizando Spring Data JPA
* Implementar autenticação e autorização
* Criar aplicações seguras com Spring Security
* Utilizar Docker para conteinerização
* Realizar deploy na AWS
* Documentar APIs com Swagger/OpenAPI
* Criar testes unitários e de integração
* Aplicar boas práticas de arquitetura e desenvolvimento
